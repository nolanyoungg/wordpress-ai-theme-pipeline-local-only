







# Nolan Young Showcase Theme X01 Blocks

















This directory contains custom block templates and styles for the Nolan Young Showcase Theme X01. These blocks are designed to enhance the theme's functionality and provide a consistent design experience across all pages.

















## Block List

















- **Hero Block**








  - Description: A premium editorial hero section with headline, supporting copy, and CTAs.








  - Usage: Suitable for the homepage and landing pages.

















- **Brand Statement Block**








  - Description: Explains the studio style and promise.








  - Usage: Suitable for the homepage and about pages.

















- **Featured Work Block**








  - Description: Displays a portfolio preview with category filter chips.








  - Usage: Suitable for the homepage and work pages.

















- **Services Preview Block**








  - Description: Provides an overview of core services with service cards.








  - Usage: Suitable for the homepage and what-we-do pages.

















- **Process Block**








  - Description: Outlines the creative process from inquiry to gallery delivery.








  - Usage: Suitable for the what-we-do pages.

















- **Signature Style / Style Pillars Block**








  - Description: Highlights key style pillars of the studio.








  - Usage: Suitable for the homepage and about pages.

















- **About Preview Block**








  - Description: Teasers a warm studio story with a CTA.








  - Usage: Suitable for the homepage and about pages.

















- **Proof or Stats Band Block**








  - Description: Displays stable stats to build trust.








  - Usage: Suitable for the homepage and about pages.

















- **Testimonials Block**








  - Description: Shows polished testimonial cards or rotators.








  - Usage: Suitable for the homepage and about pages.

















- **Resources Preview Block**








  - Description: Provides useful resource cards.








  - Usage: Suitable for the homepage and resources pages.

















- **Final CTA Block**








  - Description: A strong invitation to contact the studio.








  - Usage: Suitable for all pages.

















## Installation

















1. Ensure that the Nolan Young Showcase Theme X01 is activated in your WordPress site.








2. Navigate to the "Appearance" > "Customize" section in your WordPress dashboard.








3. Click on "Additional CSS" and add any custom styles specific to the blocks.








4. Use the block editor (Gutenberg) to insert these blocks into your pages or posts.

















## Customization

















- **Hero Block**: Customize the headline, supporting copy, and CTAs through the block settings.








- **Brand Statement Block**: Edit the text content to reflect the studio's style and promise.








- **Featured Work Block**: Add portfolio items and configure category filters.








- **Services Preview Block**: Update service card details and descriptions.








- **Process Block**: Modify the process steps to match your creative workflow.








- **Signature Style / Style Pillars Block**: Adjust the style pillars to align with your studio's values.








- **About Preview Block**: Edit the story teaser and CTA text.








- **Proof or Stats Band Block**: Enter stable stats to build trust.








- **Testimonials Block**: Add testimonials and configure display options.








- **Resources Preview Block**: Create resource cards with relevant information.








- **Final CTA Block**: Customize the CTA text and link.
