







# Icons for Nolan Young Showcase Theme X01

















This directory contains SVG icons used throughout the theme. Each icon is optimized for performance and accessibility, ensuring a consistent visual language across the site.

















## Icon Usage

















Icons are primarily used in the header navigation menu, buttons, and other interactive elements. They should be included inline within HTML or referenced via CSS to maintain scalability and performance.

















## Icon Naming Convention

















- Use lowercase filenames with hyphens for readability (e.g., `contact-us.svg`).








- Avoid using spaces or special characters in filenames.








- Ensure each icon has a descriptive name that reflects its purpose.

















## Icon Guidelines

















- **Size and Resolution**: Icons should be scalable vector graphics (SVG) to maintain quality at any size.








- **Color**: Use fill attributes for color, allowing for easy customization via CSS.








- **Accessibility**: Ensure all icons have appropriate `aria-label` attributes for screen readers. Decorative icons should use `aria-hidden="true"`.

## Adding New Icons

















1. Save the new SVG file in this directory.








2. Ensure the icon follows the naming and usage guidelines.








3. Update any CSS or HTML files that reference the icons to include the new icon.

















## Removing Icons

















1. Remove the unused SVG file from this directory.








2. Update any CSS or HTML files that reference the removed icon to remove the reference as well.

















This README serves as a guide for managing and using icons within the Nolan Young Showcase Theme X01, ensuring consistency and maintainability across the theme.
