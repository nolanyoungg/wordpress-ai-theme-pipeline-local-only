







# Nolan Young Showcase Theme X01

















## Description








Nolan Young Showcase Theme X01 is a premium, complete, classic WordPress theme designed for a photography studio website. It offers a polished portfolio, services, resources, and booking inquiry system that makes visitors trust the studio and contact it.

















## Features








- Bright editorial design with warm, refined, and premium aesthetics.








- Clean but not sterile layout.








- Image-led, confident, and human feel.








- Calm luxury, not corporate.








- High-end photography studio experience.








- Responsive design for mobile-first usage.








- Accessible interactions and focus-visible states.

## Theme Structure








The theme follows a structured directory layout to ensure maintainability and scalability. Key directories include:

















- `inc/`: Contains helper functions, setup, enqueue scripts, template tags, custom post types, and customizer settings.








- `assets/`: Holds CSS, JavaScript, icons, and images.








- `src/`: Source files for SCSS and JavaScript.








- `template-parts/`: Reusable template components.








- `page-templates/`: Custom page templates.








- `blocks/`: Custom Gutenberg blocks.

















## Assets








- **CSS:** Located in `assets/css/main.css`.








- **JavaScript:** Located in `assets/js/bundle.js`.

## Accessibility








The theme adheres to accessibility standards, ensuring that all interactions are keyboard navigable and screen reader friendly. Focus-visible states are strong, and no keyboard traps are present.

















## Installation








1. Download the theme ZIP file from the repository.








2. Upload the ZIP file via the WordPress admin panel under Appearance > Themes > Add New > Upload Theme.








3. Activate the theme.

















## Customization








- **Custom Logo:** Use the WordPress customizer to upload your studio logo.








- **Menu:** Customize the primary navigation menu in the WordPress admin panel under Appearance > Menus.








- **Colors & Fonts:** The theme uses system fonts and a predefined color palette for a consistent look. For further customization, modify the SCSS files in the `src/scss/` directory.
