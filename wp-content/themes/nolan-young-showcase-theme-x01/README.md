

# Nolan Young Showcase Theme X01





## Description


A premium, bright editorial/cinematic photography studio site with a highly interactive front page and a premium mega-header/navigation system.





## Features


- Bright editorial styling with warm whites/ivories + champagne accents + charcoal text.


- Sticky header; subtle blur or shadow on scroll; polished hover; clear focus-visible.


- Dropdown panels are solid and readable (not transparent). Warm white/ivory background, border, shadow, stable z-index.


- Mega header with left rail and right content area that updates as the user hovers AND keyboard-focuses rail items.


- Mobile-first responsive design, production-quality polish, accessible markup, visible focus states.





## Requirements


- Theme supports: title-tag, post-thumbnails, custom-logo, automatic-feed-links, responsive-embeds, html5


- Register menus: primary + footer


- Register at least one sidebar


- Enqueue assets/css/theme.css and assets/js/theme.js with filemtime cache-busting


- No jQuery dependency


- Use safe escaping and prefixed functions





## Installation


1. Download the theme from the repository.


2. Upload the `nolan-young-showcase-theme-x01` folder to your WordPress installation under `wp-content/themes/`.


3. Activate the theme from the WordPress admin panel.
